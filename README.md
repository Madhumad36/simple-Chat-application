# Simple Chat Application

## Overview
This project is a simple, fully functional web-based chat application built using HTML, CSS, and JavaScript.  
It simulates a basic chatbot experience where users can send messages and receive automated responses based on predefined logic.

The application focuses on clarity, usability, and clean design while avoiding unnecessary complexity or external dependencies.

---

## Features
- Clean and modern chat user interface
- Send messages using a button click or the Enter key
- Automated bot replies using rule-based logic
- Typing indicator for realistic interaction
- Persistent chat history using browser localStorage
- Option to clear the chat history
- Smooth message animations and improved visual hierarchy

---

## Technologies Used
- HTML5 for structure
- CSS3 for styling and layout
- JavaScript (Vanilla JS) for functionality and logic

## How It Works
1. The user types a message into the input field.
2. The message is displayed in the chat window as a user message.
3. A typing indicator appears briefly.
4. The chatbot responds based on keyword matching logic.
5. All messages are stored in localStorage to preserve chat history across page reloads.

---

## How to Run the Project
1. Download or clone the project files.
2. Ensure all three files (`index.html`, `style.css`, `script.js`) are in the same folder.
3. Open `index.html` in any modern web browser.
4. Start chatting with the bot.
